📌 Overview

This project is designed to simulate a simple response generator that provides different outputs based on user interaction. It showcases how randomness can be used to make programs more dynamic and engaging.

------------------
⚙️ Features

- Randomized responses
- User input handling
- Simple and interactive interface
- Lightweight and easy to run

----------------------
🛠️ Technologies Used

- C#
- .NET Framework / .NET Core
- Console Application

-------------------
▶️ How to Run

1. Open the project in Visual Studio
2. Build the solution
3. Run the application
4. Enter input and receive randomized responses
